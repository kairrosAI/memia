# MEMIA BPOS Agent Catalogue Notice

The MEMIA BPOS agent catalogue is included in the package.

Each agent card defines:

- a MEMIA agent name;
- a functional intent;
- expected inputs;
- expected outputs;
- an IPCRWA operating method;
- usage guardrails.

Core agents carry the `ready` status. Specialist agents carry the `included` status. Connector access, external channels and sensitive actions are controlled by the local installation and require the appropriate user approval.

MEMIA BPOS claims ownership of:

- the MEMIA agent taxonomy;
- MEMIA agent names;
- the card format;
- IPCRWA operating instructions;
- the manifest;
- CLI and MCP interfaces;
- product documentation and assets included in this package.
