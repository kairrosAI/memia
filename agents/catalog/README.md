# Agents catalog MEMIA BPOS

Ce dossier contient 200 fiches agents intégrées au package. Elles sont prêtes à lire, indexer et appeler via le CLI ou le MCP, mais elles restent désactivées par défaut.

Avant activation :

1. vérifier le besoin réel ;
2. vérifier les données manipulées ;
3. définir les permissions ;
4. définir les canaux autorisés ;
5. ajouter une procédure de rollback ;
6. conserver la notice MEMIA BPOS et le copyright.
