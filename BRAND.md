# Marque MEMIA BPOS

MEMIA BPOS est le Business and Personal Operating System de MEMIA. La marque, les logos, les assets, les noms d’agents, la taxonomie et les présentations produit restent protégés.

## Usage autorisé dans MEMIA BPOS

- afficher le logo MEMIA BPOS dans les livrables du produit ;
- utiliser la palette MEMIA pour les HTML produits localement ;
- conserver le nom MEMIA BPOS pour désigner une installation officielle ou compatible ;
- préserver les notices de copyright et d’attribution.

## Usage non autorisé sans accord écrit

- publier un service sous la marque MEMIA BPOS ;
- redistribuer les logos séparément ;
- créer une marque dérivée ou un produit commercial dérivé ;
- supprimer la notice de copyright.
